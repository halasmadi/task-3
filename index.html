I worked with a group of 4 people from my class for this project. We used IntelliJ and SceneBuilder.
We created a shopping app with both user and admin functionalities. Users can log in, browse groceries, and add items to their cart.

The app has a unique feature that compares prices from three nearby markets for the selected items. It does this by checking a database containing prices of various items across these markets. After comparing prices, the app displays to the user which nearby market offers the cheapest total price for the items in their cart. This feature helps users make cost-effective purchasing decisions while grocery shopping.

I created a mobile sensor app using Java. The app detects changes in the phone's brightness level. When the brightness decreases below a certain threshold, it triggers the display of a ghost picture on the screen.
Conversely, when the brightness increases above a specific level, the ghost picture disappears. This app essentially responds to changes in ambient light, creating a fun and interactive experience where the ghost image appears and disappears based on the surrounding brightness.
